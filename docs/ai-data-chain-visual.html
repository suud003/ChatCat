1) 架构边界图（组件视角）

Renderer 进程

  • src/chat/chat-ui.js：用户输入、消息发送、Skill 优先匹配
  • src/chat/ai-service.js：聊天系统提示词拼装、流式对话
  • src/shared/ai-client-renderer.js：聊天/记忆提取 HTTP 调用
  • src/skills/skill-router.js：/command + keyword 路由匹配
  • src/proactive/signal-collector.js：CPM/删除率/节奏信号

Main 进程

  • main.js：IPC Hub、store、技能入口、权限联动
  • src/skills/skill-engine.js：技能执行 + 上下文收集
  • src/skills/skill-registry.js：扫描 SKILL.md 并解析 frontmatter
  • src/shared/ai-client-main.js：skills/quick panel/vision 调用
  • src/quick-panel/*：润色/问答/识图业务入口

本地持久化

  • electron-store：chatHistory / aiMemories / skillsEnabled / proactiveConfig ...
  • 按日节奏：rhythmData_YYYY-MM-DD
  • 速度基线：typingSpeedBaseline
  • 文件：keyboard_YYYY-MM-DD.txt

外部模型 API（OpenAI 兼容）

  • 聊天与技能请求统一走 /chat/completions
  • 聊天多为 Renderer 直连
  • Skills / QuickPanel / OCR 多为 Main 代理

2) 业务链路（时序视角）

聊天链路（默认路径）

  1. 用户输入并发送：src/chat/chat-ui.js:725
  2. 先尝试技能匹配：src/chat/chat-ui.js:743
  3. 未命中则进入流式聊天：src/chat/ai-service.js:108
  4. 构造 system + history：src/chat/ai-service.js:116
  5. 调用 Renderer AI 客户端：src/shared/ai-client-renderer.js:93
  6. SSE chunk 实时回填 UI：src/chat/chat-ui.js:760
  7. 写回历史到 store：src/chat/ai-service.js:140

Skills 链路（命令/关键词触发）

  1. Skill 元数据加载：src/skills/skill-router.js:20
  2. 输入匹配（/command 或 keyword）：src/skills/skill-router.js:41
  3. Renderer 通过 IPC 调 Main：src/skills/skill-router.js:72 → main.js:454
  4. Main 执行技能：src/skills/skill-engine.js:30
  5. 收集上下文（converted/raw/todos/...）：src/skills/skill-engine.js:75
  6. 调用 Main AI 客户端：src/skills/skill-engine.js:47
  7. 后处理（写 convertedText / 解析 todo）：src/skills/skill-engine.js:53, src/skills/skill-engine.js:61

QuickPanel 链路（润色/问答/识图）

  1. 润色/总结/解释模板来源：src/quick-panel/text-processor.js:7
  2. 文本处理入口：src/quick-panel/quick-panel-main.js:99
  3. 问答入口：src/quick-panel/quick-panel-main.js:127
  4. 二者都走 Main 流式 AI：src/quick-panel/quick-panel-main.js:103, src/quick-panel/quick-panel-main.js:134
  5. 识图/OCR：src/quick-panel/screenshot-ocr.js:130 → src/shared/ai-client-main.js:112

记忆与行为数据链路

  1. 聊天结束后异步提取记忆：src/chat/ai-service.js:147
  2. 记忆提取系统提示词：src/chat/memory-manager.js:11
  3. 提取并持久化：src/chat/memory-manager.js:47, src/chat/memory-manager.js:125
  4. SignalCollector 计算 CPM/删除率：src/proactive/signal-collector.js:178, src/proactive/signal-collector.js:185
  5. 速度基线持久化：src/proactive/signal-collector.js:230
  6. 授权后键盘内容落盘：src/recorder/keyboard-recorder.js:251, src/recorder/keyboard-recorder.js:267
  7. 上述行为数据注入系统提示词：src/chat/ai-service.js:77
