四、MCP/Skill 接入深度
Skill优先 ▼
核心判断：Skill-first, MCP-second
MCP生态90%+面向开发者，consumer app成功案例接近零
SKILL.md（低门槛Markdown）才是ChatCat技能生态核心
防止产品变重
表层极简 — 主界面永远是猫+气泡
深层丰富 — 自然语言触发，无需理解Skill/MCP概念
渐进暴露 — 新用户3个Skill / 进阶发现更多 / 开发者导入MCP
功能预算 — 每版本最多新增2-3个Skill
版本路线
V2.0
  • 打磨现有6个Skill体验闭环
  • 不动MCP
V2.5
  • 加入日历/邮箱Skill
  • 比通用MCP更有价值
V3.0
  • UGC Skill市场开放
  • MCP精选集成(3-5个)
  • GUI包装McpConfigImporter
不建议做
✗ 通用MCP Hub（用户门槛太高）
✗ 一次性暴露所有技能
✗ MCP优先于Skill（倒置需求层级）

六、核心卖点
3大卖点 ▼
付费驱动力排名
1. 情感连接与陪伴感 ★★★★★
2. 个性化与成长感 ★★★★★
3. 实用价值(生产力) ★★★★
4. 审美与自我表达 ★★★★
5. 惊喜与探索 ★★★ | 6. 社交货币 ★★★
TOP 1：感知式陪伴 — "它知道你在忙"
不是你去找AI，而是AI感知你的状态主动出现
支撑：rhythm-analyzer + work-phase-detect + timing-judge，竞品中几乎没有
TOP 2：成长型关系 — "越用越懂你"
三个月后的猫和第一天完全不同
支撑：好感度 + 长期记忆 + 性格系统 + 惊喜事件
TOP 3：可爱的实力派 — "不只陪伴，还能帮忙"
截图解释、选中翻译、下班生成日报 — 都在桌面角落完成
支撑：Finch验证cute+useful组合（3年1000万下载）

核心竞争力公式

感知式AI（懂得什么时候出现）
× 桌宠陪伴（情感连接 + 成长感）
× 轻量生产力（截图翻译日报）
= 桌面角落的AI老朋友
最大风险："看几天就腻" — 三管齐下对策
1. AI深度（记忆+感知，每次互动不同） 2. 养成进化（好感度+毛色+功能解锁） 3. 实用价值（不只好看，还真能帮忙）
