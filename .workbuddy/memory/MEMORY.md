- ChatCat 当前最合理的产品定义是"陪伴型桌面 AI 助手"，核心不是单一桌宠或工具箱，而是桌宠情感入口 + Quick Panel 高频效率入口 + 主动交互引擎差异化 + 成长留存系统 + Runtime/Skills/MCP 平台化底座。

- 五层架构：桌面基座层 → AI Runtime 底座（三阶段收敛 + 四层Agent概念分层）→ AI 能力层 → 养成表达层 → 社交生态层。
- 版本路线：V2.0 双核进化（三Pillar+性格+记忆，~96h）→ V2.5 社交连接（串门/日历/邮箱）→ V3.0 平台生态（UGC/MCP/跨平台）。
- 北极星指标 WAU-Resident，辅助指标包括日均在线≥4h、D7留存≥35%、主动交互响应率≥20%、Quick Panel日均≥3次。
- 社交设计优先级：可晒（V2.0）→ 可陪（V2.5）→ 可创（V3.0）。
- 从 OriginMate 吸收的关键增量：兜底策略10场景、埋点20事件+3漏斗、串门社交机制、Agent提示词工程范式（Role/Inputs/ActionPath/Output/Constraints）、日历邮箱接入提前至V2.5、性格4种+演化、记忆4类+衰减。
